修改信息

Hexo基本命令
命令 含义
Hexo init 初始化博客
Hexo s 运行博客
Hexo n title 创建一篇新的文章，文章标题是title
Hexo c 清理文件
Hexo g 生成静态文件
Hexo d 部署博客（需要插件）

CMD输入 Hexo n title 创建一篇新的文章，文章标题是title
